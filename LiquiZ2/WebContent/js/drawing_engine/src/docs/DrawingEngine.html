convertCoords(el, x, y) → {Object}

Converts a given X and Y coordinates using a SVG element as reference. This method is used for snapping Links to Ports/Links, for position updates and element rotation. When linking objects, it is necessary to convert the local X and Y of a Port to a global X an Y in order to set the origin point of the Link.
Parameters:
Name Type Description
el SVGElement Instance of SVGElement
x int Original X coordinate
y int Original Y coordinate
